## ✨ Features

- **Interactive Demo**: Watch the video above to see it in action!
- **Customizable Length**: Choose a password length from 6 to 24 characters using an interactive slider.
- **Character Options**: Easily toggle between including Uppercase, Lowercase, Numbers, and Symbols.
- **Strength Meter**: Real-time password strength indicator (Weak, Medium, Strong) dynamically updates as you adjust settings.
- **One-Click Copy**: Conveniently copy your newly generated secure password to the clipboard with a single click.
- **Modern UI/UX**: Features a beautiful dark/light blue gradient background, smooth CSS transitions, and an elegant glass-like container card.

## 🚀 How to Use (Locally)

This project does not require any backend or external server. Simply download the files and run it in your favorite browser!

1. Clone the repository:
   ```bash
   git clone https://github.com/AbdulBasiTcs1/PASSWORD-GENERATOR.git
   ```
2. Open the project folder.
3. Double-click on `index.html` to open it in your browser.
4. Enjoy generating secure passwords!
